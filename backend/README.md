# Blocky Backend API using AWS Serverless REST API with DynamoDB (offline support)

## Setup

```bash
npm install
```

## Run user service offline

```bash
cd users
serverless offline start
```

## Run script service offline

```bash
cd scripts
serverless offline start
```
